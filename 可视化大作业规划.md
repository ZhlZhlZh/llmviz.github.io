# 项目核心定位与叙事主线

核心定位： 揭示论文研究热点的变化。因为LLM领域比较热门，目前先选择LLM作为研究对象（有时间可以试试接口能不能泛化到其他领域）。过去十年，LLM领域的研究热点是如何从边缘概念演变为统治级范式，并在学科间跨界迁移的？

叙事结构：**三阶段历史分期范式（基于时间序列的数据切片与交互引导）**
1. **2013–2017：表征学习与序列建模的奠基期**  （聚焦分布式词向量与 RNN/LSTM 架构的早期技术积淀，呈现研究网络的稀疏拓扑与概念萌芽）
2. **2018–2022：架构革新与预训练范式的爆发期** （以自注意力机制（Transformer）及 BERT/GPT 系列为核心，刻画模型规模化演进与引文网络的中心坍缩）
3. 2023-2026 **能力对齐、智能体协同与多模态融合期**（以 RLHF 优化、Agent 工作流及跨模态交互为特征，揭示研究生态的垂直分化与系统级能力涌现）
# 主要视图

动态力导向图+时间河流图 
- 节点=论文/技术簇，边=引文关系；河流图展示关键词频率演变。
- 滑动时间轴 → 河流图高亮当前年份 → 力导向图节点动态过滤/重排 → 中心向心力随“大爆发”年份增强。

**“蝴蝶效应”脉冲路径图**
- 点击早期节点，高亮其影响现代热点的最短引文路径，标注“知识变异”节点
- 主视图点击节点 → 左面板生成影响波路径 → 右侧详情卡显示该论文的“贡献值(Contribution Score)”。

**平行宇宙对比散点图**
- 横轴=合作广度/引用量，纵轴=结构重要性(PageRank)；颜色区分高校/企业
- 全局滑块切换维度 → 散点图实时重算映射 → 主视图节点大小/颜色同步更新

# 主要交互机制
**“时空穿越”刷选 (Temporal Brushing)**
- **实现**：D3 brush + 时间滑块。滑动时，节点按年份淡入/淡出，布局算法平滑过渡。经过2017年6月触发全屏微震动+标题浮现（CSS Animation）。
- **叙事价值**：直观呈现“从散乱到坍缩”的范式转移

**“蝴蝶振翅”溯源 (The Wing Fluttering)**
    - **实现**：预计算BFS最短引文路径。点击节点 → 脉冲动画沿边传播 → 终点节点高亮并显示贡献值（简化公式：`引用深度权重 × 结构中心性`）。
    - **叙事价值**：揭示冷门基础论文如何间接塑造现代大模型。

**“力量天平”对比 (Power Toggle)**
    - **实现**：双端滑块（高校↔企业）。切换时，D3 force simulation 重新计算节点排斥力/引力，视觉上呈现“重心偏移”。
    - **叙事价值**：让用户自主发现“算力壁垒”与“学术流量明星 vs 沉默基石”的差异。

# 建议数据格式
nodes.json
- id
- title
- year
- abstract
- authors
- citations_count
- influential_citations_count
- venue

edges.json
- source: 引用者 ID
- target：被引者 ID (父节点/基础论文)
- context(可选)**: 引用上下文文本 (如果能拿到，可用于分析“知识变异”，否则略过)。
