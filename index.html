引言

想象你是大语言模型（LLM）领域的新手研究者，面对学术网站上数量繁多的论文，我们往往会产生一些问题：
1. 整个领域有哪些细分方向或技术流派，它们是如何演进的？谁是当前的热点，又是如何突出的？
2. 众多论文里，哪些是无论什么细分方向都必读的基石论文？哪些是关键的交叉点或承上启下？
3. 当聚焦于某一篇论文时，如何快速找出它的前置、后置论文？如何按其作者等指标展开细化阅读？
因此，我们把 LLM 相关的重要论文及其对应数据（关键词、引用关系和机构地理分布等）组织成一个可探索的数据文档， 聚焦 2013-2026 年大语言模型（LLM）领域内论文研究的变化与关系，通过几个可视化图表间的相互联动，试图解决上述问题。
可以预先说明，LLM 研究不是线性增长，而是由若干关键机制反复重组形成的热点迁移。 本页面中的控件共享同一个年份和论文焦点，便于从宏观主题进入具体论文路径。

01 主题演化

基于 OpenAlex 数据库中提取的年度主题热度，本视图展示了 2013-2026 年间 AI 与 LLM 相关领域的宏观演化趋势。

  • 交互指南：拖动下方的时间滑块可缩放时间窗口；点击特定的河流分支，可高亮查看单一主题的兴衰起伏；点击右侧联动探索，将自动跳转该主题对应的相关图表，便于探索
  • 数据发现：河流的形态清晰地记录了范式的转移。可以直观地看到，早期的研究均稀疏；而在 2017 年之后，代表着 Transformer 和预训练大模型的分支开始急剧扩张，并在近两年分化出 Alignment 和 Agents 等新的磅礴支流。

02 论文结构

摒弃了传统的全局毛线球网络，本视图通过力导向图将全量论文投射到二维空间，揭示文献间的自然聚类与引用关系。

此图承接上方河流图的聚焦，并联动下方蝴蝶图使用。

  • 交互指南：支持鼠标滚轮缩放与拖拽平移。在上方搜索框输入或点击图中任意节点，系统将自动产生“引力波”效果，高亮该论文的直接关联论文，并暗化无关背景。
  • 数据发现：节点的大小代表其引用影响力。在这个“星空”中，不仅能定位到孤立的亮星，更能发现那些连接不同星团的桥接性论文。它们往往是融合了不同子领域技术的关键创新点。

04 空间格局

我们通过这一机构影响力地图，将文献元数据与真实的地理空间坐标相结合，映射出全球 LLM 研究力量的地缘分布。

  • 交互指南：地图上的气泡大小映射了该机构在选定时间范围内的核心发文/被引影响力，右侧附带实时更新的机构排行榜。
  • 数据发现：通过对比不同年份的数据，可以观察到学术重心的转移规律：在早期基础理论阶段，顶尖高校占据主导；而随着大模型步入“规模化定律”时代，拥有庞大算力资源的工业界巨头（如 Google ）在地图上的光芒逐渐掩盖了传统学术界。

05 主题流派地铁图

我们创新性地采用了“知识地铁图”隐喻，将时间线设定为 X 轴，将不同的技术流派设定为平行的 Y 轴轨道，把论文间的演进抽象为地铁线路。

  • 交互指南：点击图中的节点，不仅能高亮其所属的垂直演进路线，更重要的是观察那些跨越轨道的连线（换乘站）。结合时间滑块，您可以动态播放技术发展的历程。
  • 数据发现：本图试图解答“技术跨界融合”的问题。诸如 InstructGPT 之类的关键节点，并非只在自己的轨道上行驶，而是接收了来自底部其他轨道的连线汇聚。我们正想研究这样的继承、交叉与融合。

提交说明

设计说明与开发记录

可视化问题

本项目旨在回答：LLM 论文研究热点在 2013-2026 年间经历了哪些阶段迁移， 哪些论文承担了范式转换的桥梁作用，机构影响力又如何分布？

设计决策

主题河流用于表现关键词热度的时间连续性；力导向网络用于暴露论文引用结构； 局部路径图用于解释两篇论文之间的可达关系；地图用于补充机构空间分布。 交互采用年份过滤、节点选择、路径高亮、指标切换和按需详情，服务于“从宏观趋势到微观论文”的分析流程。

替代方案

曾考虑使用堆叠柱图替代河流图、邻接矩阵替代力导向图。最终保留河流图和网络布局， 因为它们更适合课堂演示中的趋势感知与探索式讲解；邻接矩阵可读性更稳定，但对非专业观众不够直观。

数据与引用

当前数据位于 data/processed，为课程原型阶段生成的合成论文、关键词、引用边和机构地理数据， 用于测试可视化与交互流程；世界底图使用公开 GeoJSON。若后续替换为真实论文语料，可保持同一 JSON 字段结构直接接入。

开发流程

预计开发约 30-40 工时。耗时最多的环节是数据结构设计、跨图联动、交互状态调试和移动端布局。 项目适合 1-4 人小组分工：数据清洗、可视化实现、叙事文案、部署与演示视频。

部署方式

本项目无需定制服务器，可直接通过 GitHub Pages 从仓库根目录部署。源代码、静态数据和说明文档均随页面发布。
