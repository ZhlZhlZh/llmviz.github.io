# 图表字段与视觉编码说明

本文档描述项目中每个可视化图表所使用的数据字段及其视觉编码方式。

---

## 数据文件结构

### nodes.json（论文节点）

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | Semantic Scholar 论文哈希 ID |
| `title` | string | 论文标题 |
| `year` | number | 发表年份 |
| `abstract` | string \| null | 摘要 |
| `authors` | string[] | 作者列表 |
| `topic` | string[] | 学科分类 |
| `keywords` | string[] | 关键词 |
| `link` | string | Semantic Scholar 链接 |
| `citations_count` | number | 被引次数 |
| `venue` | string | 发表会议/期刊 |
| `institution` | string[] | 作者所属机构 |

### edges.json（引用关系）

| 字段 | 类型 | 说明 |
|------|------|------|
| `source` | string | 引用方论文 ID |
| `target` | string | 被引方论文 ID |

### keyword_trends.json（主题趋势）

| 字段 | 类型 | 说明 |
|------|------|------|
| `keyword` | string | 研究主题关键词 |
| `year` | number | 年份 |
| `count` | number | OpenAlex 年度作品数 |

### institutions_geo.json（机构地理）

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 机构唯一标识 |
| `institution` | string | 机构名称 |
| `city` | string | 所在城市 |
| `country` | string | 所在国家 |
| `lat` | number | 纬度 |
| `lng` | number | 经度 |
| `community` | string | 研究社区（"english" / "chinese"） |
| `org_type` | string | 机构类型（"university" / "company" / "research_lab"） |
| `papers_count` | number | 论文数量 |
| `citations_count` | number | 总被引次数 |
| `influence_score` | number | 综合影响力评分（0–100） |

### institution_aliases.json（机构别名）

| 字段 | 类型 | 说明 |
|------|------|------|
| `canonical` | string | 规范化机构名称 |
| `aliases` | string[] | 该机构的其他名称变体 |

## 图表视觉编码

### 01 主题河流图（Theme River）

**数据来源：** `keyword_trends.json`、`nodes.json`

| 数据字段 | 视觉通道 | 编码方式 |
|----------|----------|----------|
| `year` | X 轴位置 | 线性映射到水平时间轴 |
| `count` | 河流条带厚度（Y 方向） | 堆叠面积图，每个关键词的 count 决定其条带高度 |
| `keyword` | 颜色（色相） | 按总量排名从 30 色调色板中依次分配 |
| 聚焦关键词 | 透明度 | 选中主题 opacity=0.7，其余 opacity=0.12 |
| 年份焦点 | 垂直参考线 | 当前选中年份处绘制竖线 |
| 排名信息 | 工具提示 | 悬停显示关键词名、年份、count 值和当年排名 |

**交互：** 点击河流条带聚焦单一主题；拖动年份范围滑块筛选时间区间。

---

### 02 论文力导向图（Paper Force）

**数据来源：** `nodes.json`、`edges.json`

| 数据字段 | 视觉通道 | 编码方式 |
|----------|----------|----------|
| `citations_count` | 节点半径 | 对数缩放：`3.6 + t × 10.5`（t 为归一化后的 log 值） |
| `year` | 节点填充色 | CSS 类映射三个阶段：≤2017 基础期、≤2022 爆发期、>2022 智能体期 |
| `source`/`target`（edges） | 连线 | 节点间直线表示引用/关联关系 |
| `title` | 文字标签 | 截断至 38 字符，根据标签模式决定是否显示 |
| 选中节点 + 一阶邻居 | 光晕效果 | 选中节点和邻居高亮，其余节点变暗 |
| `authors`、`institution`、`keywords` | 搜索/筛选 + 详情面板 | 用于文本搜索匹配和详情展示 |
| `venue` | 详情面板文字 | 显示发表会议/期刊 |
| `link` | 工具提示超链接 | 点击可跳转论文原文 |
| 节点度数（计算值） | 标签可见性阈值 | 度数 ≥8 或半径 ≥11 时始终显示标签 |
| 力导向布局 | X/Y 位置 | 斥力 + 连线吸引力 + 中心引力的物理模拟 |

**交互：** 滚轮缩放、拖拽平移、点击节点高亮邻居、拖拽节点固定位置、年份范围筛选。

---

### 03 蝴蝶影响路径图（Butterfly Path）

**数据来源：** `nodes.json`、`edges.json`

| 数据字段 | 视觉通道 | 编码方式 |
|----------|----------|----------|
| `citations_count` | 节点半径 | `6 + min(10, log₁₀(citations+10) × 1.8)`；中心节点固定 r=18 |
| `year` | 节点颜色 | 同力导向图的三阶段配色方案 |
| 引用方向（edges） | 左翼/右翼布局 | 左翼 = 中心论文引用的上游；右翼 = 引用中心论文的下游 |
| 引用深度（1–3 阶） | X 轴列位置 | 距中心越远 = 引用链越深 |
| 同层排名 | Y 轴分布 | 同一深度层内均匀垂直分布 |
| `title` | 换行文字标签 | 截断后显示在节点旁 |
| `topic`、`keywords` | 主题分类标签 | 侧边列表中显示"检索知识/推理对齐/多模态/架构效率/语言模型" |
| `institution` | 侧边列表文字 | 显示论文所属机构 |
| 父子关系 | 贝塞尔曲线 | 连接引用树中的父节点与子节点 |

**交互：** 点击任意节点将其设为新的中心论文；可调节每侧数量和引用深度；年份范围筛选。

### 04 机构影响力地图（Institution Map）

**数据来源：** `institutions_geo.json`、`institution_aliases.json`、`nodes.json`、`edges.json`、`public/world.geojson`

| 数据字段 | 视觉通道 | 编码方式 |
|----------|----------|----------|
| `lng`、`lat` | 地理 X/Y 位置 | 等距圆柱投影；锚点细线连接真实地理位置 |
| `org_type` | 点形状 | 圆形=公司、方形=大学、三角形=研究实验室 |
| `community` | 颜色（社区模式） | 蓝色=英文社区、红色=中文社区 |
| `org_type` | 颜色（类型模式） | 橙色=公司、绿色=大学、紫色=研究实验室 |
| `influence_score` / `citations_count` / `papers_count` | 点大小 | 用户可切换；线性映射到 4–11px 半径 |
| `institution` | 文字标签 | 高影响力或当前论文相关机构显示名称 |
| 机构间共引关系（计算值） | 连线 | 线宽与共同引用次数成正比（0.8–4px） |
| 场景指标 | 排行榜侧栏 | "谁在主导"用 influence_score、"论文背后"用 papers_count、"谁连接主线"用 link_count |
| 当前论文的机构 | 描边高亮 | 选中论文对应机构的点加粗描边 |
| `city`、`country` | 详情面板 + 工具提示 | 显示地理信息 |
| 碰撞避让布局 | 位移 + 锚线 | 重叠点自动分散，细线连回真实地理锚点 |

**交互：** 滚轮缩放、拖拽平移、切换颜色/大小维度、切换问题场景、年份范围筛选。

### 05 主题流派地铁图（Metro Map）

**数据来源：** `nodes.json`、`edges.json`

| 数据字段 | 视觉通道 | 编码方式 |
|----------|----------|----------|
| `year` | X 轴位置 | 线性映射到水平时间轴 |
| `title`、`abstract`、`topic` | 地铁线路分类 | 正则匹配将论文归入 1–3 条主题线路 |
| 主线路（primary） | Y 轴位置 | 7 条固定水平轨道，每条线路一个 Y 坐标 |
| 线路颜色 | 站点填充色 | 每条线路有固定颜色（表示学习=灰、Transformer=青、规模化=橙等） |
| 多线路归属 | 站点大小 | 换乘站 r=7.5，普通站 r=5.2 |
| `citations_count` | 排序优先级 + 标签可见性 | 高引用论文优先显示且始终带标签 |
| `source`/`target`（edges） | 换乘连线 | 跨线路的引用关系绘制为站间连线 |
| `title` | 站点标签 | 截断至 28 字符；换乘站和高引用论文显示 |
| 场景模式 | 排序策略 | "范式迁移"按引用量、"换乘站"按线路数、"近期分化"按年份 |

**7 条地铁线路：**
- 表示学习（灰色 #64748b）
- Transformer（青色 #176b87）
- 规模化预训练（橙色 #d97706）
- 对齐与偏好（红色 #dc2626）
- 多模态（紫色 #7c3aed）
- 检索与智能体（绿色 #16a34a）
- 高效架构（青绿 #0f766e）

**交互：** 点击站点选中论文；切换问题场景；年份范围筛选。

### 06 LLM 关键节点时间线（Mini Timeline，页面顶部）

**数据来源：** 内置于 `src/modules/llm-mini-timeline/data.js`（静态数据，改编自 [llm-timeline](https://github.com/Michaelgathara/llm-timeline)）

| 数据字段 | 视觉通道 | 编码方式 |
|----------|----------|----------|
| `year` + `month` | X 轴位置 | yearFraction = year + (month-1)/12，线性映射 |
| `branch` | Y 轴车道位置 | 10 条分支各占一个水平车道 |
| `branch` 对应颜色 | 圆点填充色 + 连线描边色 | 每个分支有固定颜色 |
| `parentIds` | 贝塞尔曲线连线 | 父→子节点间绘制曲线表示演进关系 |
| `title` | 原生 SVG tooltip | 悬停显示标题和年月 |
| `description` | 详情卡片文字 | 点击后在下方卡片中展示 |
| `modelSize` | 详情卡片元数据 | 显示参数规模 |
| `link` | 详情卡片超链接 | 可跳转论文/资源 |
| 选中状态 | 高亮/变暗 | 选中节点的祖先和后代链高亮，其余变暗 |

**10 条分支：** Foundation（蓝）、Decoder-Only（红）、Encoder-Only（黄）、Encoder-Decoder（绿）、Mixture-of-Experts（紫）、Open-Source（蓝）、Alignment（橙）、Theory（青）、Multimodal（紫）、Hybrid（灰）

**交互：** 点击圆点查看详情并高亮关联链路；点击空白区域取消选中。

## 全局联动机制

所有图表共享一个全局状态（`app-state.js`），包含以下字段：

| 状态字段 | 说明 |
|----------|------|
| `yearRangeStart` | 年份范围起点 |
| `yearRangeEnd` | 年份范围终点 |
| `year` | 当前焦点年份（通常等于 yearRangeEnd） |
| `selectedPaperId` | 当前选中论文 ID |
| `selectedInstitutionId` | 当前选中机构 ID |
| `selectedTheme` | 当前选中主题关键词 |

每个图表通过共享的双滑块年份范围筛选器（`year-range-filter.js`）与全局状态双向同步：
- 任一图表修改年份范围 → 全局状态更新 → 其他图表自动响应
- 右侧导航栏实时显示当前全局年份范围、选中主题和选中论文
